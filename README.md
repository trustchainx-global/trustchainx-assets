## English

This repository hosts **public brand assets** for TrustChainX (logos, icons, banners, product screenshots).  
It does **not** contain application source code.

### Structure
```
/brand/
  logos/
  icons/
  banners/
  brand-guidelines.md
/product/
  screenshots/
  renders/
/source/                # Original design files (tracked via Git LFS)
```

### Usage
- For official press or partnership use, please follow the rules in **brand/brand-guidelines.md**.
- Do not modify, distort, or recolor the logo except as allowed in the guidelines.
- Commercial or public use requires permission — email **legal@trustchainx.io**.

### Security
If you find a security issue related to these assets or their distribution, report it per **.github/SECURITY.md**.
